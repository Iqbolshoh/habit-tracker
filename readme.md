# 🧠 Habit Tracker (Vanilla JavaScript)

A simple, modern and responsive **Habit Tracker web application** built using HTML, CSS, and JavaScript.

This project helps users build better daily habits by tracking progress and marking completed tasks in a clean and minimal UI.

## 🎯 Project Purpose

The main goal of this project is to practice and demonstrate:

- DOM manipulation using JavaScript
- Local storage usage in real applications
- Clean UI/UX design principles
- Responsive layout development
- Real-world frontend logic implementation

---

## ✨ Features

- 📝 Add new habits
- ✔ Mark habits as completed / undo
- 💾 Data saved in LocalStorage
- 📊 Progress visualization
- 🎨 Modern glassmorphism UI design
- 📱 Fully responsive layout
- ⚡ Smooth animations and interactions

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + modern UI design)
- JavaScript (Vanilla JS)
- LocalStorage API
